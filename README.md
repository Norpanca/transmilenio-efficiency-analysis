# 🚌 TransMilenio Efficiency & Mobility Analysis
### Urban Data Science Project

---

## 📊 Project Overview
This analysis evaluates the operational efficiency of the TransMilenio system in Bogotá. By processing high-volume transit data, I identified bottlenecks in specific corridors and analyzed passenger demand patterns to suggest optimizations.
